## 操作说明:

### 移动控制:
- 键 "W": 物体前移（Z轴负方向）
- 键 "S": 物体后移（Z轴正方向）
- 键 "A": 物体左移（X轴负方向）
- 键 "D": 物体右移（X轴正方向）
- 键 "Q": 物体上移（Y轴正方向）
- 键 "E": 物体下移（Y轴负方向）

### 旋转控制:
- 键 "↑" (上方向键): 物体绕X轴正向旋转
- 键 "↓" (下方向键): 物体绕X轴负向旋转
- 键 "←" (左方向键): 物体绕Y轴负向旋转
- 键 "→" (右方向键): 物体绕Y轴正向旋转
- 键 "Z": 物体绕Z轴正向旋转
- 键 "X": 物体绕Z轴负向旋转

### 缩放控制:
- 键 "+" (等号键): 放大物体
- 键 "-" (减号键): 缩小物体

### 其他:
- 键 "Esc": 退出程序
